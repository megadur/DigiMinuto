Allgemeine Hinweise

Die folgenden Hinweise geben einen einfachen Überblick darüber, was mit Ihren personenbezogenen Daten passiert, wenn Sie diese Website besuchen. Personenbezogene Daten sind alle Daten, mit denen Sie persönlich identifiziert werden können.

Die Datenverarbeitung auf dieser Website erfolgt durch den Websitebetreiber. Dessen Kontaktdaten können Sie dem Impressum dieser Website entnehmen.

Wir hosten unsere Website bei GitHub Pages. Anbieter ist die GitHub, Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA. Wenn Sie unsere Website besuchen, erfasst GitHub verschiedene Logfiles inklusive Ihrer IP-Adressen.

2. DigiMinuto App und Nostr-Netzwerk

Die DigiMinuto-Anwendung (App) für Android, iOS oder Windows arbeitet vollständig dezentral. Weder der Websitebetreiber noch die App sammeln, speichern oder verarbeiten personenbezogene Daten auf zentralen Servern.

Wenn Sie die "Senden via Nostr"-Funktion nutzen, kommuniziert die App mit öffentlichen Relays im Nostr-Netzwerk. Diese Kommunikation erfolgt dezentral. Bitte beachten Sie, dass IP-Adressen von den Betreibern der jeweiligen Relays erfasst werden können, auf die wir keinen Einfluss haben.

3. Kontakt

Wenn Sie uns per E-Mail kontaktieren, wird Ihre Anfrage inklusive aller daraus hervorgehenden personenbezogenen Daten (Name, Anfrage) zum Zwecke der Bearbeitung Ihres Anliegens bei uns gespeichert und verarbeitet. Diese Daten geben wir nicht ohne Ihre Einwilligung weiter.
